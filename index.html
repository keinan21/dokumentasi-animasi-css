Transitions Dasar

CSS Transitions memungkinkan perubahan nilai properti CSS dilakukan secara bertahap (smooth) selama durasi waktu tertentu.

Transisi sangat berguna untuk efek hover, perubahan warna, ukuran, dan lain-lain tanpa perlu JavaScript.

Ada dua hal penting saat menggunakan transition: properti yang akan berubah, dan nilai akhirnya.

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

Contoh shorthand untuk transition:

transition: background-color 0.3s ease-in-out;

Contoh lengkap penggunaan transition:

button {
  background-color: #8c5ea1;
  color: white;
  transition: background-color 0.3s ease;
}
button:hover {
  background-color: #b799d6;
}

Gunakan transition untuk menciptakan efek visual yang lebih menarik dan halus pada elemen halaman webmu.

Keyframes Animation

CSS Animations memungkinkan kita membuat efek visual kompleks menggunakan @keyframes untuk menentukan langkah-langkah animasi.

Animasi tidak butuh interaksi pengguna, bisa dimulai otomatis saat halaman dimuat.

  1. @keyframes mendefinisikan tahapan animasi
  2. Properti animasi ditambahkan ke elemen target

Properti umum dalam animasi:

  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction

Contoh penggunaan @keyframes dan animasi:

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
.box {
  animation-name: fadeIn;
  animation-duration: 2s;
}

Animasi cocok untuk banner, efek masuk elemen, dan loader sederhana.

Perbedaan Transition dan Animation

Meskipun terlihat serupa, transition dan animation memiliki perbedaan mendasar:

  • Transition butuh interaksi pengguna, animation bisa berjalan otomatis
  • Transition hanya dari A ke B, animation bisa banyak tahap
  • Animation pakai @keyframes, transition tidak

Gunakan transition untuk efek ringan dan animation untuk efek lebih kompleks.

transition: all 0.3s ease-in; animation: slideIn 1s forwards;
Timing Functions CSS

Timing function menentukan kecepatan transisi atau animasi berubah selama durasi waktu.

  • linear - kecepatan konstan
  • ease - lambat di awal dan akhir
  • ease-in - lambat di awal
  • ease-out - lambat di akhir
  • cubic-bezier - fungsi kustom

Contoh penggunaan timing function:

transition: transform 0.5s ease-in-out; animation-timing-function: cubic-bezier(0.68, -0.55, 0.27, 1.55);

Timing yang tepat bisa membuat animasimu terasa lebih alami.

Efek Hover dengan Transition

Efek hover adalah salah satu penggunaan paling umum dari CSS transition.

Contoh umum: mengubah warna tombol saat disentuh kursor.

a {
  color: #8c5ea1;
  transition: color 0.3s ease;
}
a:hover {
  color: #b799d6;
}

Kamu juga bisa menerapkan efek pada ukuran, bayangan, rotasi, dan opacity.

  • Efek zoom dengan transform: scale
  • Efek fade dengan opacity
  • Efek geser dengan translate
